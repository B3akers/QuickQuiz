# QuickQuiz

QuickQuiz to wieloosobowa gra quizowa online, umożliwiająca szybkie i emocjonujące rozgrywki z przyjaciółmi lub społecznością Twitch. Gra została stworzona z myślą o polskich użytkownikach.

## Strona internetowa

[https://quickquiz.ovh](https://quickquiz.ovh)

## Funkcje

- Logowanie jako gość lub przez konto Twitch
- Tworzenie własnego lobby z możliwością konfiguracji:
  - liczby rund
  - limitu czasu na pytanie
  - wyboru kategorii
- Zapraszanie znajomych do lobby
- Rozgrywka w czasie rzeczywistym dla wszystkich graczy
- Tablica wyników po zakończeniu gry

## Jak grać

1. Wejdź na stronę: [https://quickquiz.ovh](https://quickquiz.ovh)
2. Zaloguj się jako gość lub przez Twitch
3. Stwórz lobby i dostosuj ustawienia gry
4. Zaproś znajomych i rozpocznij rozgrywkę
5. Odpowiadaj na pytania i rywalizuj o najwyższy wynik

## Technologie

Projekt został zrealizowany przy użyciu:

- Svelte 5 – frontend
- ASP.NET Core (C#) – API
